数独ソルバーの使い方

本アプリは，スマートフォン，タブレット，PCのブラウザで手軽に動作する，1画面統合型の数独解答・思考支援ツールです．自動で盤面を埋めてしまうだけではなく，論理的に確定可能な数字をヒントとして提示し，ユーザー自身が考えて解き進めるためのチュートリアル用に最適化されています．

◆ 画面構成と特徴

  • 盤面の統合: 出題（問題入力）と解答が同一の盤面上で行われます．
  • 文字色のルール: 最初に入力された数字（問題）は赤色で固定され，ユーザー自身が入力した解答や自動計算による解答は黒色で表示されます．
  • ユーザー解答モード: Solveボタンを押すと，解答入力を開始します．このボタンを押した時点の盤面が初期配置（問題）として固定されます．また，クロスハッチングなどの論理によって一意に確定するマスを自動で検索し，マジェンタ（赤紫色）で表示します．
  • 思考支援ヒント: Nextボタンを押した際，確定可能な数字がマスの隅に小文字の青色で表示されます．
  • 自動計算モード: Ansボタンを押すと，バックトラック・アルゴリズムが内部で高速計算を行い，現在の盤面から最後まで一気に解いた完全な解答を出力します．また，新たに埋まった解答データ（例：127 132 146）が上の入力欄に自動出力されます．

◆ 問題・解答の入力方法

画面下部のテキストボックス，または画面上の数字ボタンを使って入力・変更を行います．

  1. ボタン入力（個別入力）: 「行（縦）」「列（横）」「数字」の順番で3桁の数字を入力し，OKボタンを押します．
    （例：1行2列目に「3」を入れる場合 → 1 2 3 と連続して押し，OKをクリック）
    ※ 複数のマスを連続して入力することも可能です（例：123 456 と押して OK を押すと，2マス分が同時に反映されます）．
  2. コピペ入力（一括入力）: テキストボックスにキーボードで直接文字を入力するか，外部のテキストデータを貼り付けて OK を押すと，一括で盤面に反映されます．
  3. 修正と消去: 入力途中で間違えた場合は BS（バックスペース）ボタンで1文字消去できます．すでに盤面に入力された数字を消去したい場合は，数字として「0」を入力して OK を押してください．
    （例：1行2列目の数字を消去して空欄に戻す場合 → 1 2 0 OK）

◆ 各種操作ボタンの機能

  • Reset : 盤面, 内部の入力履歴，ログをすべて消去し，初期状態に戻します．
  • Next : 論理的に絞り込めるマスを自動解析し，そのマスに入る可能性のある候補の数を青色で小さく表示します．
    なお，ルール違反や矛盾があるルートに入り込んだ場合はエラーメッセージが表示され，自動的に1手前の状態に巻き戻ります（ロールバック機能）．
  • Back : ご自身の操作を1手前に戻します．試行錯誤（分岐）の途中で行き詰まった際に便利です．
  • Rec : 解答データの読み込み（復元）を行います．保存しておいた解答データ（例：127 132 146）を下のメッセージ欄にペーストしてこのボタンを押すと，盤面に黒色で解答が反映されます．問題（赤，マジェンタ）と解答（黒）を区別した状態で盤面を復元する際に便利です．

◆ 超難問への挑戦（AI Escargot）

2006年にフィンランドの数学者 Arto Inkala 氏によって考案され，当時「世界一難しい数独」として大きな話題を呼んだ「AI Escargot（エスカルゴ）」という問題です．人間が解くには高度な仮説と先読みが必要ですが，本ソルバーの「前処理（マジェンタによる確定）」と「Ans（バックトラック全探索）」を組み合わせることで，AIがどのようにこの難問を瞬時に攻略するかを体感できます．

Sudoku Solver (Beginner / Tutorial Edition)

This application is a single-screen integrated Sudoku solving and cognitive assistance tool optimized for mobile and desktop browsers. Instead of automatically filling in the board, it provides logical candidates as hints, making it an excellent tutorial tool for users to think and solve puzzles by themselves.

◆ Layout and Design Features

  • Single Grid Integration: Both puzzle configuration and solving take place on the same grid layout.
  • Color-Coded Feedback: The initial puzzle numbers are locked and displayed in red, while user inputs and the AI's solutions are shown in black.
  • Cognitive Hinting: When you click the Next button, logically deducible numbers appear as tiny blue digits in the corners of cells.

◆ Input Methods (Puzzles and Answers)

You can enter data using the text box or the number buttons located below the grid.

  1. Button Input (Cell-by-Cell Entry): Press the buttons in the sequence of "Row number", "Column number", and "Target digit", then click OK.
    (Example: To place "3" in Row 1, Column 2 → Continuously press 1 2 3, then click OK)
    * You can also enter data for multiple cells at once (e.g., entering 123456 and clicking OK updates two cells simultaneously).
  2. Paste Input (Bulk Entry): Type directly into the text box using your keyboard, or paste copied data from external files and click OK to fill the grid instantly.
    * It supports both continuous 3-digit sets and an 81-character continuous string of digits (use "0" for empty cells) with multi-line support.
  3. Correction and Erasure: Press the BS (Backspace) button to delete the last typed character. To clear a number already placed on the grid, enter "0" for that cell and click OK.
    (Example: To clear Row 1, Column 2 → Input 1 2 0 and press OK)

◆ Functions of Control Buttons

  • Reset : Clears the entire grid, user history, and internal logs, resetting the application to its initial state.
  • Solve : Enters the user-solving mode. The current configuration is captured as the puzzle and locked in red.
  • Next : Dynamically analyzes the grid to find logically deducible cells and displays their candidates in small blue text.
    * Note: In this Beginner Edition, the AI will NOT automatically fill the cells. Use the blue hints as a guide to deduce why that number fits and enter it manually.
    If a rule violation or logical contradiction occurs along your chosen path, an error message appears, and the grid automatically rolls back to the last valid state.
  • Back : Reverts your last action by one step. Ideal for backing out of an incorrect branch during trial-and-error.
  • Ans : Triggers a powerful backtracking algorithm that computes the final solution instantly from the current state.
  • Rec : Outputs your full solving history and operation logs as raw text data in the message field.
【Recommended Demonstration Scenario】
Set a puzzle, press Solve, and proceed using Next. On difficult puzzles, the logical hints will pause at branching options. Take a guess and manually enter a digit. If you make a wrong choice, the AI's Next analyzer will detect the contradiction downstream and automatically guide you back—a perfect demonstration of human-AI collaboration.

\newslide{オプションとデータ名}

\vspace*{18mm}

\slidepage
\begin{itemize}
\item
Reference(以下Ref)のP11---12を参照
\item
コマンドの引数の最後に\verb|[ ]|をおく
\item
省略したらデフォルト値を使う\\
例)\verb|Listplot("1",[[p1,p2]],["dr,2","Color=red"]);|\\
例)\verb|Plotdata("2","sin(5*x)","x",["Num=200"]);|
\item
データの名前は，種別＋最初の文字列\\
　上の例の場合のデータ名は　sg1, gr2
\end{itemize}
